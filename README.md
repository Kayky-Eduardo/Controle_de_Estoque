O Sistema de Gerenciamento de Recursos Humanos em discussão, tem como finalidade fornecer uma solução completa e adaptável para a gestão de recursos humanos em diferentes tipos de empresas.

O sistema permite, dentre suas principais funcionalidades:

· Controle de ponto eletrônico;

· Gestão de funcionários;

· Controle de cargos e níveis de acesso;

· Gestão de banco de horas;

· Gestão de folha de pagamento;

· Gestão de férias e licenças;

· Geração de relatórios e métricas.

O projeto foi concebido como um sistema modular e configurável, permitindo que o contratante defina regras de jornada, pausas, setores e níveis de acesso conforme sua necessidade.
